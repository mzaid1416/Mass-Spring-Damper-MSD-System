# Mass-Spring-Damper-MSD-System
Quarter-Car Suspension System simulation using MATLAB and Simulink. The model analyzes vehicle displacement, velocity, and suspension response to sinusoidal road input using a Mass-Spring-Damper system. The project demonstrates dynamic system modeling, vibration analysis, and vehicle suspension performance.
This project focuses on the modeling and simulation of a Quarter-Car Suspension System using MATLAB and Simulink. The quarter-car model is a simplified representation of a vehicle's suspension system that consists of a sprung mass, a spring, and a damper. It is widely used in mechanical and automotive engineering to study the vertical motion of a vehicle under different road conditions while reducing the complexity of analyzing a complete vehicle. The mathematical model of the suspension system was developed using the governing differential equations of motion and implemented in Simulink through standard simulation blocks. MATLAB was used to define the required parameters, including the sprung mass, spring stiffness, damping coefficient, and simulation time. This project demonstrates how mathematical equations can be transformed into a graphical simulation model to analyze the dynamic behavior of engineering systems accurately and efficiently.
The simulation was implemented using commonly available Simulink blocks such as Sine Wave, Derivative, Gain, Sum, Integrator, Mux, and Scope. Two different simulation models were developed to evaluate the suspension system from different perspectives. The first model analyzes the vehicle displacement and velocity responses under a sinusoidal road disturbance, while the second model compares the road input with the vehicle displacement to study the vibration isolation capability of the suspension. The generated Scope outputs were analyzed to observe transient and steady-state responses, oscillation characteristics, system stability, and the influence of damping and spring stiffness on the overall suspension performance. The results confirmed that the mathematical model behaved as expected and remained dynamically stable throughout the simulation.
This project provides practical experience in MATLAB programming, Simulink modeling, dynamic system simulation, and mechanical vibration analysis. It demonstrates the importance of simulation in engineering by allowing system performance to be evaluated without constructing a physical prototype, thereby reducing development time, cost, and experimental risks. The project also enhances the understanding of vehicle dynamics, feedback systems, and suspension design principles, making it a valuable learning resource for students, researchers, and beginners in mechanical and automotive engineering. Furthermore, the developed model can serve as a foundation for future studies involving active and semi-active suspension systems, different road profiles, parameter optimization, and advanced control techniques for improving ride comfort, vehicle stability, and overall suspension performance.
